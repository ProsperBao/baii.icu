### 问题

```typescript
type case1 = Reverse<['a', 'b']> // ['b', 'a']
type case2 = Reverse<['a', 'b', 'c']> // ['c', 'b', 'a']
```

### 解答

```typescript
type Reverse<T> = T extends [...infer Rest, infer L]  ? [L, ...Reverse<Rest>] : T
```

1. 利用 `infer` 拆分数组
