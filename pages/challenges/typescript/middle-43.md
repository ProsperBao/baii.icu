### 问题
实现一个类型版本的 `Array.shift`

```typescript
type Result = Shift<[3, 2, 1]> // [2, 1]
```

### 解答

```typescript
type Shift<T> = T extends [infer _, ...infer Rest] ? Rest : never
```
